# JS Tile & Grout Cleaning Services

![Astro](https://img.shields.io/badge/Astro-5.17-BC52EE?logo=astro&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.4-06B6D4?logo=tailwindcss&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-Strict-3178C6?logo=typescript&logoColor=white)

Sitio web estático para **JS Tile & Grout Cleaning Services**, empresa de restauración de pisos que sirve al sur de Florida. Construido con Astro, Tailwind CSS y TypeScript.

## Stack Tecnológico

| Tecnología | Versión |
|:-----------|:--------|
| Astro | 5.17 |
| Tailwind CSS | 3.4 |
| TypeScript | Strict |
| Inter + Open Sans | Google Fonts |

## Primeros Pasos

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo (localhost:4321)
npm run dev

# Build de producción
npm run build

# Previsualizar build de producción
npm run preview
```

## Estructura del Proyecto

```
src/
├── pages/               # Páginas del sitio
│   ├── index.astro      # Página principal
│   ├── aboutUs.astro    # Sobre nosotros
│   ├── services/        # Páginas de servicios
│   ├── serviceAreas.astro
│   ├── gallery.astro
│   └── contact.astro
├── components/          # Componentes reutilizables
├── layouts/             # Layout base
└── data/                # Datos de servicios, galería y áreas
```
